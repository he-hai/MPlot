`MPlot.py` is a python script for ploting data from Mass Photometry (Refeyn).  

The repository contains also demos.  

The script has been tested in an environment with following packages:  
- python 3.8.2
- numpy 1.23.4
- pandas 1.3.4
- matplotlib 3.5.1
- seaborn 0.11.2
- lmfit 1.0.0
- openpyxl 3.0.6
- scipy 1.4.1
